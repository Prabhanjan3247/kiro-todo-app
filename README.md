# 📝 Kiro To-Do List Web App

A simple and interactive **To-Do List web application** created using **Kiro AI IDE**.
This project demonstrates how AI-assisted development can quickly generate a functional web application.

---

## 🚀 Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Simple and responsive user interface
* Runs directly in the browser

---

## 🛠️ Technologies Used

* **HTML** – structure of the web page
* **CSS** – styling and layout
* **JavaScript** – functionality and interactivity
* **Kiro AI IDE** – used to generate and assist with the project code

---

## 📂 Project Structure

```
kiro-todo-app
│
├── index.html     # Main webpage
├── style.css      # Styling for the app
└── script.js      # JavaScript functionality
```

---

## ▶️ How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Double-click **index.html** or open it in any web browser.

The To-Do List application will start running locally.
